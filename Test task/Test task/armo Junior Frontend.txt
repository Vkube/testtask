Написать на Electron.js приложение, которое запрашивает эти данные и отображает их в своем интерфейсе;
Написать скрипт в виде отдельного файла, который скачает необходимые зависимости (tornado, electron и др.), а затем запустит сервер и приложение; 
Выложить код на GitHub и прислать ссылку на репозиторий.
Дополнительно в письме укажите сколько времени потребовалось на выполнение задания, а также с какими трудностями столкнулись.
Тестовое задание будет проверяться на чистой виртуальной машине Windows 10 с последними версиями Node.js LTS и Python 3.
